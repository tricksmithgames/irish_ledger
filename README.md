# Irish Ledger

AI-powered Irish digital newspaper with games, community content, and bilingual support.

## Vision

A modern digital newspaper for the Irish diaspora, powered by AI agents and featuring:

- **Craicle** - Daily word game (think Wordle meets Irish craic)
- **Community content** - Crowdsourced photos and stories from readers
- **Bilingual** - English and Irish (Gaeilge)
- **AI-powered newsroom** - Agent-based editorial workflow

## Agent Architecture

| Agent | Role |
|-------|------|
| Editor-in-Chief | Story priorities, content review, editorial voice |
| Beat Reporters | Research & write stories (culture, sports, diaspora) |
| Social Scout | Monitor social media for trending topics & user content |
| Translator | English ↔ Irish translation |
| Puzzle Master | Create daily Craicle and other games |
| Newsletter Curator | Assemble daily/weekly newsletters |
| Community Manager | Reader engagement |
